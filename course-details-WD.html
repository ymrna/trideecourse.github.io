Web Development

Ada banyak pilihan untuk pelatihan pengembang online, tetapi kursus ini tidak diragukan lagi yang paling komprehensif dan efektif di pasaran.

Apa yang akan Anda pelajari?

HTML5, CSS3, dan JavaScript Fundamental

Dalam kursus ini kita akan belajar tentang:

  1. Apa yang dilakukan HTML/CSS/JS?.

  2. Dasar dasar HTML/CSS/JS/CSS.

  3. Langkah dan semantik selanjutnya.

  4. Framework CSS dan JS.

  5. Memperkenalkan DOM.

Layout yang responsif, mudah diakses, dan indah.

Dalam kursus ini kita akan belajar tentang:

  1. Flexbox dan Responsive Design.

  2. JavaScript (all 2020 modern syntax, ES6, ES2018, dll.).

  3. Asynchronous JavaScript - Promises, async/await, dll.

  4. AJAX and single page apps.

  5. Bootstrap 4 and 5 (alpha).

  6. Semantic UI dan Bulma CSS Framework.

Membuat website portofolio HTML dan CSS statis dan landing page.

Dalam kursus ini kita akan belajar tentang:

  1. DOM Manipulation.

  2. Unix(Command Line) Commands.

  3. NodeJS,NPM,ExpressJS.

  4. Templating dan REST.

  5. SQL vs. NoSQL databases.

Membuat aplikasi task list animasi.

Dalam kursus ini kita akan belajar tentang:

  1. MongoDB.

  2. Cookies & Sessions.

  3. Authorization.

  4. Schema Design,Mongoose,Authentication From Scratch.

  5. Database Associations

Membuat aplikasi web full-stack dari awal.

Dalam kursus ini kita akan belajar tentang:

  1. Common Security Issues - SQL Injection, XSS, dll.

  2. Developer Best Practices.

  3. Deploying Apps dan Cloud Databases

  4. Image Upload dan Storage.

  5. Maps dan Geocoding.
